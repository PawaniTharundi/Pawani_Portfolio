SDGP Project Posted on 15th March 2026

Building PetHub My Biggest Project Yet

As part of my Software Development Group Project (SDGP) in Level 5, our team built PetHub — an AI-powered smart pet care management platform. This was by far the most ambitious and rewarding project I have ever worked on.

What PetHub Does

PetHub is a full-stack platform that brings together several features in one place:

  • 🐾 Pet Health Tracking — owners can log health records and vet visits
  • 🛒 Verified Marketplace — a trusted space to buy/sell pet-related products
  • 🤖 AI Chatbot — gives personalised recommendations and answers questions
  • 📍 Stray Animal Reporting — community-powered reporting system

Tech Stack I Used

React.js Node.js MongoDB Atlas Firebase Machine Learning

What I Learned

This project taught me how to work in a real team using Git and GitHub for version control, how to integrate machine learning models into a web app, and how to deploy a full-stack application using Vercel and Railway. It also pushed me to improve my communication skills as we worked in sprints with regular team meetings.

The most challenging part was the AI chatbot integration — connecting the ML model to the REST API and making it respond in real time. But seeing it work in the final demo was incredibly satisfying!

Personal Project Posted on 20th August 2025

Life Below Water My First Full Website with HTML & CSS

During my Level 4 studies, I built a fully responsive website called Life Below Water as a personal project. The site focuses on ocean sustainability and marine life protection — inspired by the UN Sustainable Development Goal 14.

Project Overview

The website was built entirely with HTML5 and CSS3, with no JavaScript frameworks. This was a fantastic exercise in mastering the fundamentals of web development before moving on to React.js in later projects.

  • 🌊 Multiple pages with consistent navigation
  • 📱 Fully responsive using media queries (mobile, tablet, desktop)
  • 🎨 Custom colour palette inspired by ocean blues and greens
  • 🖼️ Image galleries and informational sections about marine life

Key Skills I Practised

HTML5 Semantics CSS3 Flexbox Media Queries Responsive Design CSS Animations

Reflection

This project gave me confidence that I could build something complete and polished from scratch. It validated my decision to study Computer Science and pushed me to go deeper into frontend development. Looking back at it now, after learning React.js and building PetHub, I can see how much I've grown as a developer.

It also taught me the importance of planning — I sketched wireframes before writing a single line of code, and that saved me a lot of time during implementation.
